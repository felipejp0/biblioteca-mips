# Sistema de Gerenciamento de Biblioteca em Assembly MIPS

Trabalho prático da disciplina de Arquitetura de Computadores.

## Funcionalidades

- Cadastrar livro
- Emprestar livro
- Devolver livro
- Listar livros cadastrados
- Encerrar o sistema

## Como executar

O código foi desenvolvido em Assembly MIPS e pode ser executado no simulador MARS.

Arquivo principal: `biblioteca.asm`
